My Linux Story by Prithvi Vishak

Who can contribute and how?

Anyone can contribute. Though this site is aimed at Linux users residing in Mysore, anyone from anywhere is welcome to write about their interests. Our GitHub page has more detailed instructions and guidelines on the same.

This site is hosted on what used to be an old Toshiba laptop, now given a facelift and christened "PAC" from the initials of its creators. It sports an Intel Pentium T2060 (32-bit) and a whopping 3GB of RAM. It runs the Apache 2 HTTP server on openSUSE Tumbleweed.
